Modpack customizado para o servidor de Minecraft S.I.N.C. (Sigla Inventada Nada Criativa) 2.5.

## Requisitos
- Minecraft 1.20.1
- Forge 47.4.20
- Java 17+
- 4GB+ de RAM recomendados

## Instalação
Este modpack é distribuido de duas maneiras:
* Archive .zip com os arquivos brutos ('bruto')
* Archive .zip em formato CurseForge para instalação automática ('curseforge')

### Arquivos Brutos
1. Crie uma instância que cumpra os [Requisitos](https://github.com/void-v0id/sinc-modpack#requisitos)
2. Baixe a versão mais recente em [Releases](https://github.com/void-v0id/sinc-modpack/releases)
3. Descompacte na pasta '.minecraft/' em sua instância
4. Execute a instância

### ZIP CurseForge
1. Baixe a versão adequada em [Releases](https://github.com/void-v0id/sinc-modpack/releases)
2. Procure pela opção de 'Import' em seu launcher de preferência (Prism Launcher/ATLauncher têm maiores chances de sucesso)
3. Execute a instância que foi instalada para você
